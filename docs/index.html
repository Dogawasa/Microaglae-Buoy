System Architecture

  • Upper Liquid3-style sealed photobioreactor for safe microalgae growth
  • Lower water treatment chamber with intake, filter media, and outlet flow
  • Oxygen is routed through tubing, check valve, and diffuser into the water
  • ESP32 sensor cluster sends live values to the AI backend
  • AI chooses TREAT, FLUSH, HOLD, or LOCKOUT

Prototype Build

  • Floating frame or pontoons
  • Clear upper Liquid3-inspired algae photobioreactor
  • Lower treatment box with tubing and pump
  • Waterproof electronics box with battery and relay

Recommended Extra Hardware

  • Waterproof enclosure and cable glands
  • Fuse and 12V to 5V buck converter
  • Intake mesh and algae-retaining membrane
  • Leak sensor, ballast, and manual switch

How to use this site

  1. Deploy the Flask backend somewhere public or on your local network.
  2. Paste the backend base URL above and click Save.
  3. Open this GitHub Pages site from any browser.
  4. Use simulation buttons or connect a real ESP32 module.

API routes used by this frontend

  • GET /data
  • POST /simulate
  • POST /manual_lock
